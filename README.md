# Тести з законодавства України

Веб-застосунок для підготовки до іспиту на знання законодавства. **589 питань** у 4 розділах з відстеженням прогресу.

## Можливості

- **Режим тесту** — питання з прихованими відповідями, підрахунок балів, розбір помилок
- **Режим вивчення** — список питань або картки зі свайпом/клавіатурою
- **Випадковий тест** — 20 довільних питань з розділу для швидкого повторення
- **Прогрес** — зберігається у `localStorage`: найкращий результат, кількість спроб, смуга днів
- **Темна тема** — автоматично за системними налаштуваннями
- **Мобільний** — оптимізовано під iPhone (safe area, PWA-ready)

## Розділи

## Структура

```
quiz-app/
├── index.html      # Single-page shell
├── app.js          # Логіка (vanilla JS, ~1000 рядків)
├── data.js         # База питань (589 питань)
├── style.css       # Стилі + темна тема
└── netlify.toml    # Заголовки кешування
```

Без фреймворків, без збірки — відкрити `index.html` у браузері достатньо.

## Запуск локально

```bash
# Будь-який статичний сервер, наприклад:
npx serve .
# або
python3 -m http.server 3000
```

## Деплой

Застосунок задеплоєний на [Netlify](https://netlify.com). Будь-який push в `main` → автоматичний деплой.

## Технічний стек

- Vanilla JS (ES6+), без залежностей
- CSS custom properties (підтримка `prefers-color-scheme`)
- `localStorage` для прогресу (схема v2 з міграцією з v1)
- SVG-іконки inline
